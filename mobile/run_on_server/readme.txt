Example for Genero 3.0 EAP using the new RunOnServer frontcall

Before running the program, please amend the 3 CONSTANTS to suitable values
for your environment.

The IPADDRESS is the address where the Genero Application Server is running
The PORT value should be 6394 by default
The APP value can be any app you have run via GAS that has created a URL in APPDATA/app
